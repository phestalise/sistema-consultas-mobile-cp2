# 📱 Sistema de Consultas Mobile

Aplicativo mobile desenvolvido com **React Native + Expo** para gerenciamento de consultas médicas.

---

## 🚀 Tecnologias

* React Native
* Expo
* TypeScript
* Node.js

---

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/sistema-consultas-mobile-cp2.git
```

Entre na pasta:

```bash
cd sistema-consultas-mobile-cp2
```

Instale as dependências:

```bash
npm install
```

---

## ▶️ Como rodar o projeto

Inicie o servidor:

```bash
npx expo start
```

Depois:

* 📱 Baixe o app **Expo Go** no celular
* Escaneie o QR Code exibido no terminal ou navegador
